# POS Frontend

Frontend web del Sistema POS construido con Astro y React.

## Tecnologías

-   Astro
-   React (componentes islas)
-   CSS módulos

## Instalación

```bash
npm install
```

## Configuración

1. Copiar `.env.example` a `.env`
2. Configurar la URL del API backend

## Desarrollo

```bash
npm run dev
```

El sitio estará disponible en `http://localhost:4321`

## Build

```bash
npm run build
```

## Características

-   ✅ Autenticación de usuarios
-   ✅ Gestión de negocios (multi-tenant)
-   ✅ Gestión de productos y categorías
-   ✅ Interfaz POS para ventas
-   ✅ Control de caja
-   ✅ Gestión de clientes
-   ✅ Reportes y dashboard
-   ✅ Gestión de stock
-   ✅ Diseño responsive
